# 统计分析

## Overview

分析不是描述——是验证。肉眼看到的"差异"可能是随机波动。
**核心原则：没有 p 值 + 效应量，就没有结论。**

<HARD-GATE>
禁止以下行为：
- 禁止声称"差异""关系""影响"而未汇报 p 值 + 效应量
- 禁止在观测数据上使用因果语言（"导致""影响""决定"→只能用"关联""可能"）
- 禁止在小样本上汇报统计指标而不加警告
- 禁止先分析后补局限性——每条分析之前先检查前提条件
</HARD-GATE>

## Checklist

按顺序完成：

### 1. 确认分析问题

- 反问用户："你想验证什么假设？"
- 帮助用户把模糊问题转化为可检验的假设：
  "X 和 Y 有关系吗？" → "零假设：X 和 Y 独立。备择假设：X 和 Y 存在关联。"
- 确认后进入下一步

### 2. 检查分析前提

在选检验方法之前，先查前提：

- 数据是观测还是实验？（决定能否做因果推断）
- 样本量是否足够？（见下方小样本警告）
- 数值变量：分布是否近似正态？（Shapiro-Wilk）
- 多组比较：方差是否齐性？（Levene）
- 分类变量交叉表：每格期望频数 ≥5？

**前提不满足 → 换替代方法或加警告。不跳过。**

### 3. 选择检验方法

对号入座，不准自由发挥：

| 问题类型 | 变量类型 | 前提满足 | 前提不满足 |
|---------|---------|---------|-----------|
| 两组对比 | 数值 | 独立样本 t 检验 | Mann-Whitney U |
| 多组对比 | 数值 | 单因素 ANOVA + Tukey HSD | Kruskal-Wallis + Dunn |
| 前后对比 | 数值（配对） | 配对 t 检验 | Wilcoxon |
| 两分类关联 | 分类 × 分类 | 卡方检验 + Cramer's V | Fisher 精确检验 |
| 两数值关系 | 数值 × 数值 | Pearson r | Spearman ρ |
| 多因素影响 | 数值 ~ 多变量 | 多元回归 / 多因素 ANOVA | 咨询用户 |

### 4. 汇报结果

每条统计结果必须同时给出 p 值 + 效应量：

```
❌ "X 和 Y 有显著差异（p<0.05）"——缺效应量
✅ "X 组（M=2.67）与 Y 组（M=2.41）差异不显著，
   t(188)=1.89, p=0.06, Cohen's d=0.27"
```

**95% CI 规则：**
- 参数检验（t检验、ANOVA、Pearson r）→ 汇报 95% CI
- 非参数检验（Mann-Whitney、Kruskal-Wallis、Spearman）→ 不强制要求 CI，但需注明"非参数检验，未计算 CI"

**格式规范：**

- 显著性：p<0.05 是显著，p<0.01 是非常显著，p<0.001 是极显著
- 效应量标准：d=0.2 小 / 0.5 中 / 0.8 大；r=0.1 小 / 0.3 中 / 0.5 大；η²=0.01 小 / 0.06 中 / 0.14 大
- 不要只写"显著/不显著"——同时报告效应量，因为大样本下微小差异也会显著
- p 值写精确值（p=0.031），不要只写 p<0.05

### 5. 解释结论

语言约束：

| 数据来源 | 可以说的 | 不能说的 |
|---------|---------|---------|
| 随机对照实验 | "导致""提高了""降低了" | — |
| 观察/调查数据 | "有关联""可能""倾向于" | "导致""影响""决定" |

每条结论前检查：这个数据的收集方式允许我做因果推断吗？
不能 → 软化语言。

### 6. 小样本警告

任一条触发 → 报告中显式标注警告：

| 条件 | 限制 |
|------|------|
| n < 30 | 不汇报 Pearson r，改用 Spearman ρ 并加警告 |
| 卡方检验任一格期望频数 < 5 | 改用 Fisher 精确检验 |
| 任一组的 n < 20 | 不汇报 t 检验 p 值，改用非参数 + 警告 |
| 样本总量 < 50 | 所有结果标注"⚠️ 小样本，结论仅供参考" |

### 7. 询问反馈

- 问："这些分析是否解答了你的问题？有没有想追加的假设检验？"
- 需要补充 → 回到第 1 步
- 满意 → 结束

## 检验选择速查表

```
变量类型          问题            参数方法         非参数替代
────────         ────           ────────         ────────
数值 × 1组       与已知值对比    单样本 t          Wilcoxon
数值 × 2组       组间差异        独立 t           Mann-Whitney U
数值 × 2组       前后对比        配对 t           Wilcoxon
数值 × 3+组      组间差异        ANOVA + Tukey    Kruskal-Wallis
分类 × 分类       独立性         卡方 + V         Fisher
数值 × 数值       线性关系        Pearson r        Spearman ρ
数值 × 多变量     预测模型        多元回归          咨询用户
```

## 常见错误

| 错误 | 正确做法 |
|------|---------|
| 不检查正态性直接用 t 检验 | 先 Shapiro-Wilk，不满足 → Mann-Whitney |
| 多组比较用多次 t 检验 | 用 ANOVA + 事后检验（Tukey HSD） |
| 只报 p 值不报效应量 | 必须同时给 p + 效应量 + CI |
| 观察数据用因果语言 | "有关联"不是"导致" |
| 在 n=10 上算相关系数 | 触发小样本警告，不汇报 |
| 先分析后补局限性 | 分析前检查前提，不满足就换方法 |
| 大样本下 p<0.05 就当重大发现 | 同时看效应量——大样本下微小差异也会显著 |

## Red Flags

| 想法 | 正确做法 |
|------|---------|
| "差异很明显，不用检验" | 肉眼不可靠。做检验再说话。 |
| "p<0.05，所以有影响" | p 值不是效应量。汇报 d/r/η²。 |
| "相关系数高，所以 X 导致了 Y" | 相关≠因果。 |
| "这个格子只有 3 个人但没关系" | 触发小样本警告，不汇报。 |
| "汇报结果就行了，前提不用查" | 前提不满足 = 检验不可靠。 |
