# statistical-analysis Demo

## 方法：TDD for Skills

## RED 阶段（无 Skill）

**素材来源：** data-visualization 报告中的统计错误

**5 个错误：**
1. 没做假设检验就说"差异"
2. 在 n=10 上算相关系数（-0.45），统计上无意义
3. 汇报相关系数但没给 p 值和置信区间
4. 观察数据用因果推断语言（"暗示""影响"）
5. 局限性放在最后当免责声明，而非用在分析前过滤

## GREEN 阶段（加载 Skill）

**三个假设全部合规：**
- ① 就业方向-GPA：Shapiro 不通过→自动切 Kruskal-Wallis，H=8.34, p=0.50, ε²<0.01
- ② 专业-就业：卡方 χ²(63)=68.98, p=0.28, V=0.099
- ③ 奖学金-GPA：Mann-Whitney U, p=0.58, r=−0.02

全部附带 p 值 + 效应量，观测数据用"关联"语言。
