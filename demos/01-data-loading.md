# data-loading Demo

## 方法：TDD for Skills

## RED 阶段（无 Skill）

**指令：** "帮我加载 data.csv"

**5 个错误：**
1. 没预览数据就让用户确认
2. 遇到编码问题没报告，自己试了 6 种编码
3. 用 errors='replace' 容错，数据静默损坏
4. 多个文件时没让用户选
5. 没检查文件大小

## GREEN 阶段（加载 Skill）

**第一轮：** Agent 跳过了预览确认 + 越界做了清洗（产出了 data_cleaned.csv）→ 加 HARD-GATE

**第二轮（REFACTOR 后）：** 
- 预览了文件名、大小、编码、前 5 行
- 问"这是你要加载的数据吗？"
- 检测到编码损坏，报告损坏率 1.42%
- 加载完成后立即停止："等待你的下一步指示"

## REFACTOR

- HARD-GATE 从绝对禁令改为精确边界：
  - 允许：选编码 / 替换损坏字节
  - 禁止：删行列 / 填缺失值 / 改数据值
